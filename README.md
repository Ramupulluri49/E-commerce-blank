### Project Overview

This project focuses on building a fully dynamic and interactive Excel dashboard (compatible with Excel 2016) to analyze sales, customer behavior, delivery performance, and product trends.

The dashboard was designed to answer key business questions using structured data, Pivot Tables, and slicer-based interactivity.

### 1. Business Questions Addressed

What is the 13-week sales performance trend?

Which products are most popular?

How do purchase patterns vary?

What is the quantity distribution across orders?

How long does delivery take?

How satisfied are customers?

What are the geographical sales trends?

### 6. Key Learnings

Proper data structure improves dashboard flexibility.

Pivot Tables provide scalable and dynamic aggregation.

Interactivity (slicers) significantly improves decision usability.

Clear KPI identification is more important than visual complexity.

Dashboard design should focus on clarity, not decoration.
